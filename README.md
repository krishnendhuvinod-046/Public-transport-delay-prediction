# Public Transport Delay Prediction

This project analyzes and predicts public transport delays using machine learning.

## Technologies Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Scikit-learn

## Key Insights
- Traffic congestion significantly increases delay times.
- Rainy weather leads to higher average delays.
- Peak hours show the highest delay patterns.

## Machine Learning Model
Random Forest Regressor was used to predict delay minutes.
